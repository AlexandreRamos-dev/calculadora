#Calculadora Simples

Este é um projeto de uma **Calculadora Simples** que realiza operações básicas como adição, subtração, multiplicação e divisão. O objetivo deste projeto é fornecer uma ferramenta simples para demonstrar operações matemáticas fundamentais.

#Funcionalidades

A calculadora suporta as seguintes operações:

- Soma (+)
- Subtração (-)
- Multiplicação (×)
- Divisão (÷)
